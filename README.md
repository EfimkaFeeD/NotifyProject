# NotifyProject

## Концепт и идея

По скольку многие (музы)кальные сервисы требуют денег за подписку, мы создадим бота, который будет имитировать приложение, похожее на Яндекс.Музыку или же Spotify.\
Также, потому что мы можем, мы интегрируем ChatGPT и Stable Diffusion, чтобы пользователи могли наслаждаться разработками умных людей (не нашими).

## Структура проекта

    ├─ source/              Исходный код приложения
    │  ├─ discord/          Исходный код для бота Discord
    |  |  ├─ assets/        Материалы для бота Discord
    |  |  ├─ utils/         Функции для бота Discord
    │  ├─ telegram/         Исходный код для бота Telegram
    |  |  ├─ assets/        Материалы для бота Telegram
    |  |  ├─ utils/         Функции для бота Telegram
    ├─ telegram.py          Запуск бота для Telegram
    └─ discord.py           Запуск бота для Discord

## Планируемые возможности

* Для начала, выбор языка
* В главном меню будет:
  * Музыка
    * Плейлист "избранное"
    * Поиск трека по названию
    * ??? Волна треков
  * ChatGPT
    * Возможность задать вопрос, и даже с какой-то вероятностью получить ответ
    * Сохраненные ответы (до 50 штук)
  * Stable DIffusion
    * Возможность сгенерировать картинку по текстовому запросу
    * Сохраненные картинки (до 5 штук)
